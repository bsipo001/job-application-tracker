


# Job Application Tracker

A web application to track job applications - COP4751 Course Project

A web application to help track job applications during the job search process.



\## Description

This is a web application built using Flask, MySQL, and HTML/CSS to track job applications.



\## Features

\-Manage companies

\-Track job listings

\-Track applications and status

\-Store contacts

\-Job match feature based on skills


## Technologies 

\-MySQL Database

\-Python with Flask

\-HTML/CSS for the web interface



\## Setup Instructions



\### 1. Clone the repository

```bash

git clone https://github.com/bsipo001/job-application-tracker.git

cd job-application-tracker




### 2. Create Virtual environment:
-Run the following commands


python -m venv venv

venv\\Scripts\\activate



\### 3. Install Dependencies:

pip install -r requirements.txt

### 4. Setup environment variables



Create a .env file:



DB\_HOST=localhost

DB\_USER=jobuser

DB\_PASSWORD=your\_password

DB\_NAME=job\_tracker



\### 5. Run the application

python app.py





6\. Open in browser

http://127.0.0.1:5000
